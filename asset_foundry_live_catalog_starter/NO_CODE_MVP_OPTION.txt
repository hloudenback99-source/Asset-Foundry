NO-CODE MVP OPTION
==================

If the API setup feels too technical today, launch the branded HTML using the sample JSON file first.

How:
1. Edit public/assets.sample.json manually whenever an asset changes.
2. Host the static site on Vercel, Netlify, or even your existing site if it supports HTML uploads/embeds.
3. Use Notion only as your internal database until you're ready to connect the API.

This means:
- No Notion token needed yet.
- No API setup needed yet.
- You still get the branded premium front end.
- You manually update the asset data until API connection is worth setting up.

Best approach:
Launch static first, get buyer feedback, then connect Notion once the structure is proven.
